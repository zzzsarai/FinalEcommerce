# E-Commerce Web Application

A simple e-commerce web application where users can browse products, add them to a cart, and manage products through an admin dashboard.

## Description

This project allows users to view a catalog of products, add or remove items from a shopping cart, and see the total price. Admins can add, edit, or delete products and upload images. The app is fully responsive and works on both desktop and mobile screens.

## Technologies Used

- **Frontend:** React, React Router, HTML, CSS  
- **Backend:** Laravel (PHP)  
- **Database:** MySQL  
- **Others:** Fetch API, FormData for image uploads

## Features

- Product catalog with images, names, descriptions, and prices  
- Shopping cart with quantity controls  
- Admin dashboard to manage products  
- Image upload with live preview  
- Responsive design
